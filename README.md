# auc_bot
A discord bot made in discord.py for the Antistasi Ultimate Community.

# FIRST TIME SETUP:
`python3 -m pip install -U discord.py`
or
`py -3 -m pip install -U discord.py`

### Manual Requirements:
Create a `bot_token.py` file and inside it add these 2 variables:
```py
bot_token = BOT_TOKEN
debug = False
```
